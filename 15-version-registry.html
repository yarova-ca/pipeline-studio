How to use this page — version maintenance workflow

  1. Find the item whose version you want to verify (use the category sections below).
  2. Click the source link → check the latest release on that page.
  3. If the version differs: find the doc(s) listed in the "Used in" column → update the version string there.
  4. Update the "Last verified" date here to today's date.
  5. For base images: also update the Dockerfile FROM tag in doc 11 and any framework-specific Dockerfiles.
  6. Commit with message: Update [item name] version [old] → [new] (verified [date])
