How to read each industry card — 6 sections explained

Systems & Products: Named software systems, platforms, or products actually built in this industry in Canada. Not generic categories — specific named things.

Project Types: Type of tech work that is common (modernization, greenfield, compliance, data pipelines, mobile, embedded, etc.).

Employers by Province: Named Canadian employers, grouped by province. Only provinces with significant industry presence shown.

Tech Stack Signals: Technologies that appear frequently in job postings for this industry in Canada. Not exhaustive — the dominant signals.

Compliance & Regulatory: Named Canadian and sector-specific regulations. Each entry: regulation name — plain-English description of what it requires.
